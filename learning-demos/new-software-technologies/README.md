
```js
npm i express sequelize dotenv bcryptjs mysql2 nodemon
```

### Disable `any` Warnings in TypeScript

Update ESLint config:

```json
{
  // ...
  "@typescript-eslint/no-explicit-any": "off"
  // ...
}
```
